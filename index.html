fox

A (simple!) package manager to install your tools with ease
think Homebrew, but for private (and public) repos

        sudo curl -fsSL "install.getfox.sh" | bash

Open-source MIT Licensed.

See detailed installation instructions

Multiplatform

Works on unix like systems; linux and macOS. No need to use a different package manager on every system that you use to get the tools that you need.

ðŸ’Š Doctor command

Use the built-in doctor command to check for problems as well as recommendations.

Autocompletion

Use the built-in completion command to generate auto-completions for various shells; fish, bash, zsh.

Interactive search

Search for packages with a nice graphic TUI. No need to install unofficial third-party apps to get a nice UI for your installed packages.

Install from private repos

Easily share your private tools with your colleagues. Make your packages installable with one copy-paste command.

Hand curated packages

There is an official repository with hand-cured packages that you can install right away.
